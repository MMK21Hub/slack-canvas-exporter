Hackatime API

Hackatime docs made by the community since official sucks

feel free to edit!

​

Authentication

all authentication must be provided as a header 

Authorization: Bearer API_KEY

To find your API key, go to https://hackatime.hackclub.com/my/wakatime_setup and copy HACKATIME__API__KEY

For example:

 Authorization: Bearer 854ede89-d6cc-4d7c-9ba3-279ff68d472d (Random Key)

Routes

Retrieve User Trust Factor

https://hackatime.hackclub.com/api/v1/users/{SLACK_ID}/trust_factor

Returns the trust factor of a user which can be

Green

Trusted/Community Leader

​

Blue

Normal Person

​

Red

Banned

​

Get current user's today time (By API Key)

https://hackatime.hackclub.com/api/v1/users/current/statusbar/today

​

Returns

data → grand_total → text and total_seconds

{"data":{"grand_total":{"text":"Start coding to track your time","total_seconds":0}}}

Get Another User's Time Today by Slack ID (Requires API Key)

https://hackatime.hackclub.com/api/v1/users/SLACK_ID/statusbar/today

For Example, 

curl -H "Authorization: Bearer API-KEY" \ https://hackatime.hackclub.com/api/hackatime/v1/users/SLACk_ID/statusbar/today

Returns

data → grand_total → text and total_seconds

{"data":{"grand_total":{"text":"Start coding to track your time","total_seconds":0}}}

Get current user's stats for a specific time range (By API Key)

    https://hackatime.hackclub.com/api/v1/users/my/stats

URL Parameters

start_date (optional, in format YYYY-MM-DD)

end_date (optional, in format YYYY-MM-DD)

features (optional, known features: languages and projects, defaults to languages)

​

Example:

https://hackatime.hackclub.com/api/v1/users/my/stats?start_time=2025-09-01&end_time=2025-12-01&features=projects,languages

Get raw heartbeat data for the current user (By API Key)

https://hackatime.hackclub.com/api/v1/users/my/heartbeats

URL Parameters

limit (optional, max 100): Amount of heartbeats to get

start_date (optional, in format YYYY-MM-DD)

end_date (optional, in format YYYY-MM-DD)

Example:

curl https://hackatime.hackclub.com/api/v1/users/my/stats?start_time=2025-09-01&end_time=2025-12-01&limit=100
