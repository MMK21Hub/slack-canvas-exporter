How do I link my scene to another:question:

insert a link into your HTML! This is done using

<a href="/scenes/number of connected scene/">Name of connected scene</a>

​

What even is a scene:question:

A scene is a simple web page using HTML, CSS and JS (JS is optional). It should have background art for the relevant place on the map and a fun little story along with links to the surrounding scenes, you can see an example on node 19

File ID: F09E71N73PT File URL: https://hackclub.slack.com/files/U09FFCBKK47/F09E71N73PT/image.png

what are those arrows:question:

They show directional links! Arrows mean a link exists from A to B, a path means a link that goes both ways.

​

What are the submission requirements:question:
All scenes need to have a background, HTML content styled with CSS, a fun bit of a story or lore and links to all connecting scenes.

How do I claim:question:

Post in the thread with your plans for the scene and the repository where you will work! The scene will be assigned to the first person with a working demo.
If a scene is claimed with no demo, it can be re-assigned.
