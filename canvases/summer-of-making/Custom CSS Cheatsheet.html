Custom CSS Cheatsheet

This document provides you with a way to customize your SoM profile page however you want! Anyone can contribute to this document! : )

Customizing your profile picture's size

The following code customizes your profile picture:

.profile-avatar {
  width: 300px;
  height: 300px;
}

This code sets your profile picture to be 300 pixels in width and height.

​

Additionally, if you want your profile picture to look square:

border-radius: 25px;

sets the border-radius of your profile picture to only 25 pixels, making it a rounded square.

You can also change the image and size with this:

.profile-avatar { display: none; }

.profile-header::before {
  content: "";
  display: block;
  
  width: 256px;
  height: 256px;
  
  background-image: url("https://placehold.co/256x256/png");
  background-size: cover;
  background-position: center;
  
  border-radius: 100%;
  
  margin-left: auto;
  margin-right: auto;
  margin-bottom: 1rem;
}

Customizing the "Activity" header

The following code targets the "Activity" header on your profile:

h2.text-2xl.font-bold,text-gray-800 {
  font-size: 30px;
  font-weight: 700;
}

You see, font-size: 30px; sets the size of the heading to 30px!

​

Customizing the background of the page with an image

The following code targets the body element on the page and sets it to an image:

body::before {
  background: url("https://wallpapercrafter.com/th800/32857-Moraine-Lake-Banff-Canada-mountains-forest-4k.jpg");
  background-size: cover;
  background-repeat: no-repeat;
  background-position: center center;
}

It uses body::before as the Summer of Making website uses it to set the background image of the web pages. Thus, to override their image, you must use body::before.

​

Customizing the background of cards on the profile page

"Cards" refers to the rectangles that contain information on your profile, including devlogs, the sidebar that holds your profile picture, name, stats, etc.; your badges and projects.

The following code sets the background of these cards:

.card-with-gradient {
  background: rgba(255, 255, 255, 0.7) !important;
}

.card-with-gradient refers to the cards mentioned above. !important is required as without it, Summer of Making's default style will take precedence and apply itself over your styles. This will set the background of your cards to be a translucent white.

Additionally, if you want a hover effect on these cards:

.card-with-gradient:hover {
    background: rgba(255, 255, 255, 0.9) !important;
}

This will do the trick. It just makes the cards more opaque. Again, !important is required because without it, not only will Summer of Making's default style will take precedence and apply itself over your styles, but this style may also get overridden by the other style!
